# Java with Dagger2 
Sample java code with dagger

 

## This project uses :
* Gradle
* JUnit

##Build
To build, build from root project 

`./gradlew build`

##Run
To run, run from root project (*just a `System OutPrint()` ~*)

`./gradlew run`

##Testing
To run unit test, run this

`./gradlew test`
